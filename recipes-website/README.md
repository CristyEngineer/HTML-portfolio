# 🎂 Mom's Best Recipes

A warm and simple recipe website dedicated to family traditions and homemade dishes.  
This project is built with pure HTML and CSS as part of my HTML portfolio.

## 📁 Project Structure

- **index.html** – Homepage  
- **recipes.html** – Overview of all recipes  
- **recipe-detail-pasta.html** – Bolognese pasta recipe + video  
- **recipe-detail-wraps.html** – Chicken wraps recipe + video  
- **recipe-detail-soup.html** – Mom’s soup recipe + video  
- **submit-recipe.html** – Form to submit your own recipe  
- **blog.html** – Small kitchen stories using `<details>`  
- **about.html** – About the project and its purpose  

## ✨ Features

- Clean and semantic HTML  
- Consistent navigation across all pages  
- Simple responsive layout using CSS  
- `<details>` and `<summary>` for collapsible blog stories  
- Recipe detail pages with images and video tutorials  
- A functional recipe submission form (frontend only)  

## 🛠 Technologies Used

- **HTML5**  
- **CSS**  
- No backend  

## 🚀 How to View

Open `index.html` in your browser, use VScode or host the project using GitHub Pages.

## 📌 Notes

This project focuses on structure, accessibility, and clean HTML.  
It is designed as a portfolio piece to demonstrate frontend fundamentals.
